## Pre-requisites

#### Node - [Nodejs](https://www.nodejs.org)

## How to run

Clone the repo

run `npm install`

run `npm run build`

run `npm start`

Browse to http://localhost:3000

***On slow networks CRUD features may not work properly***
    
To run Tests

switch to [Test](https://github.com/xXZang3tsuXx/mern-task-listtree/tests) Branch

clone the repo

`npm install`

`npm test`
